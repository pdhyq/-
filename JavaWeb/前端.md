# HTML基础标签

```html
<html>
    <head>
        <title>这是我的第一个程序</title>
        <meta charset="UTF-8"> 
    </head>
    <body>
        hello <br/>world;
        <p>这里是一个段落</p>
        <p>这是第二个段落</p>
        <img src="OIP-C.jfif" width="200" height="200" alt="这里是一张图片">
        <h1>标题一</h1>
        <h2>标题一</h1>
        <h3>标题一</h1>
        <h4>标题一</h1>
        <h5>标题一</h1>
        <h6>标题一</h1>
        <ol type="A" start="3">
            <li>你好</li>
            <li>世界</li>
            <li>再见</li>
            <li>
                <ul type="square">
                    <li>hello</li>
                    <li>world</li>
                    <li>!!!!!</li>
                </ul>
            </li>
        </ol>
        <b><i>你好</i>世界，<u>hello world</u></b>
        <br/>
        水分子化学式：H<sub>2</sub>O<br/>
        a的平方：a<sup>2</sup><br/>
        5&gt;4<br/>
        <a href="http://www.baidu.com">百度一下</a>
    </body>
</html>
<!--
html语言是解释型语言，不是编译型
浏览器是容错的

html页面由一对标签组成：<html></html>
<html>称为开始标签
</html>称为结束标签

title表示网页的标题
可以在mate标签中设置编码方式
<br/>表示换行。br标签是一个单标签。单标签表示开始和结束标签是同一个，就像<br></br>，于是把斜杠放在后边
<p> 表示段落标签

img 标签图片文件
    src属性表示图片的路径
    width 和 height 表示图像的大小
    alt 表示图片的提示

路径：  
    相对路径  
    绝对路径

标题标签
    h1~h6

列表标签：
    - ol 有序列表
        start 表示从*（阿拉伯数字）开始，type表示类型：A a I i 1（default）
    - ul 无序列表
        type disc（default）circle square
    
u 下划线 b 粗体 i 斜体

实体标签，大于小于号什么的

span 不换行的块标记

a表示超链接
    href 链接的地址
    target：
        _self  在本窗口打开
        _blank 在一个新的窗口打开
-->
```
